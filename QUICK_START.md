# Field Service Suite Quick Start

This is the fast onboarding guide for daily use.

## 1. Sign In

### Staff

1. Open the app URL.
2. Click `Login`.
3. Enter your username/password.

### Client

1. Open the app URL.
2. Click `Client Portal`.
3. Enter your customer email and password.

## 2. Staff First 10 Minutes

1. Open `Dashboard` and check job counts + alerts.
2. Open `Jobs` and review `new` and `in-progress`.
3. Open `Schedule` for today’s workload.
4. Open `Notifications` (bell icon) and clear unread items.

## 3. Most Common Actions

### Create Job

1. Go to `Jobs`.
2. Create new job.
3. Set customer, priority, schedule date.
4. Assign technician.
5. Save.

### Update Job in Field

1. Open assigned job.
2. Check in.
3. Add notes/photos/parts.
4. Check out when complete.

### Add Project Task

1. Go to `Projects`.
2. Open a project.
3. Go to `Tasks`.
4. Click `Add Task`.
5. Save.

### Update Task Planning

1. Go to `Planner`.
2. Select project.
3. Edit task dates and progress.
4. Verify status updates.

### Create Invoice

1. Go to `Invoices`.
2. Create invoice for customer/job.
3. Set status (`pending` or `paid`).

## 4. Tools

Use `Tools` menu for:

1. `Inventory`
2. `Equipment`
3. `Quotes`
4. `Export` (CSV)

## 5. Role Snapshot

1. `Admin`: Full access.
2. `Dispatcher`: Operations management access.
3. `Technician`: Assigned work updates.
4. `Client`: Own jobs/invoices only.

## 6. If Something Fails

1. Refresh page and retry.
2. Confirm you are in correct portal (staff/client).
3. Report:
   - page name
   - action taken
   - exact error text
   - time of issue
