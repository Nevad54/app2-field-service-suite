# Field Service Suite - Quick Start (Print)

Version: 1.0  
Audience: Admin, Dispatcher, Technician, Client  
Use: A4 training handout

---

## 1) Login

### Staff

1. Open app URL.
2. Click `Login`.
3. Enter username and password.

### Client

1. Open app URL.
2. Click `Client Portal`.
3. Enter customer email and password.

---

## 2) First 10 Minutes Checklist

Mark each item when complete:

- [ ] Open `Dashboard` and review overall status
- [ ] Open `Jobs` and review `new` + `in-progress`
- [ ] Open `Schedule` and verify today’s jobs
- [ ] Open `Notifications` bell and clear unread items

---

## 3) Core Daily Actions

### Create a Job

1. `Jobs` -> create new.
2. Fill customer, title, priority, schedule date.
3. Assign technician.
4. Save.

### Field Update (Technician)

1. Open assigned job.
2. Check in.
3. Add notes/photos/parts.
4. Check out on completion.

### Add Project Task

1. `Projects` -> open project.
2. `Tasks` tab -> `Add Task`.
3. Fill task details.
4. Save.

### Update Planner

1. `Planner` -> select project.
2. Update task dates/progress.
3. Confirm status is correct.

### Invoice

1. `Invoices` -> create invoice.
2. Assign customer/job and amount.
3. Update status (`pending` / `paid`).

---

## 4) Tools Menu

1. `Inventory`
2. `Equipment`
3. `Quotes`
4. `Export` (CSV)

---

## 5) Roles

1. Admin: full access
2. Dispatcher: operations management
3. Technician: assigned work updates
4. Client: own jobs/invoices only

---

## 6) Troubleshooting

If something fails:

1. Refresh and retry.
2. Confirm correct portal (Staff or Client).
3. Report:
   - page name
   - action
   - exact error message
   - time it happened
